# Swift – Quick Buy for WooCommerce

Swift adds a "Buy Now" button to your WooCommerce products. One click adds the product to the cart and sends the shopper straight to checkout, skipping the cart page.

## Features

- Buy Now button that adds to cart and redirects in a single click.
- Appears on single product pages, shop and archive loops, or both, and can sit before or after the native add-to-cart button.
- `[swift_buy_now]` shortcode to place the button anywhere, optionally targeting a product by id.
- Redirect to checkout (skip the cart) or to the cart, with an option to empty the cart first.
- Optionally respects the quantity chosen on the product page for simple products.
- Button style options (theme default, solid, outline) with an optional accent colour; stateless, with no custom tables or product meta.

## Installation

1. Upload the plugin to `/wp-content/plugins/swift`, or install it via Plugins → Add New.
2. Activate it. WooCommerce must be active.
3. Open **WooCommerce → Swift Quick Buy** to configure the label, placement and redirect target.

## Frequently Asked Questions

**Does it require WooCommerce?**
Yes. Swift needs an active WooCommerce installation.

**Does it skip the cart?**
Yes, if you set the redirect target to "Checkout". The product is added to the cart and the shopper is taken straight to the checkout page. You can also redirect to the cart instead.
